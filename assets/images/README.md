# Marceau Solutions Brand Assets

## Logo Files

### Full Logo with Text
- **File**: `marceau-logo-full.png`
- **Usage**: Website header, marketing materials, documents
- **Description**: Full Marceau Solutions logo with golden circle emblem and "EMBRACE THE PAIN & DEFY THE ODDS" tagline
- **Colors**: Gold gradient (#C9A961 to #FFD700) on dark background

### Logo Icon Only
- **File**: `marceau-logo-icon.png`
- **Usage**: Favicon, social media profile, app icons, small spaces
- **Description**: Standalone golden circle emblem
- **Colors**: Gold gradient (#C9A961 to #FFD700)

## Brand Colors

**Primary Gold**: `#FFD700`
**Dark Gold**: `#C9A961`
**Background Dark**: `#1a1a1a`
**Pure Black**: `#000000`

## Typography

**Primary Font**: Serif (for "MARCEAU SOLUTIONS")
**Tagline Font**: Serif, spaced lettering
**Body Text**: Sans-serif (Arial, system fonts)

## Usage Guidelines

1. **Minimum Size**: Logo should not be smaller than 100px wide
2. **Clear Space**: Maintain padding equal to the height of one letter
3. **Background**: Works best on dark backgrounds (black, dark gray)
4. **Do Not**:
   - Distort or stretch the logo
   - Change the colors
   - Add effects (drop shadow, glow, etc.)
   - Place on busy backgrounds

## File Formats Needed

- ✅ PNG (with transparency) - for web
- ⏳ SVG (vector) - recommended for scalability
- ⏳ High-res PNG (2x, 3x) - for retina displays
